# 🧮 Animated Calculator (Streamlit)

LIVE DEMO::https://codeallphacalculator-5fhp2mj68ijzckdzbnznnw.streamlit.app/

An interactive and visually appealing calculator built using HTML, CSS, and JavaScript, embedded into a Streamlit web application.

---

## 🚀 Features

- 🧮 Basic arithmetic operations (+, −, ×, ÷)
- 🎨 Smooth animated UI with modern design
- ⚡ Fast and responsive interface
- 🔄 Clear and reset functionality
- 🌐 Runs inside Streamlit web app

---

## 🛠️ Tech Stack

- Python  
- Streamlit  
- HTML  
- CSS  
- JavaScript  

---

calculator-streamlit/

## ⚙️ How It Works

- The calculator UI is created using HTML, CSS, and JavaScript  
- It is embedded into the Streamlit app using `components.html()`  
- JavaScript handles calculations using dynamic input

## 🌍 Deployment

You can deploy this project using:

- Streamlit Community Cloud (Recommended)

### Steps:
1. Upload project to GitHub  
2. Go to Streamlit Cloud  
3. Click **Deploy App**  
4. Select your repository  
5. Done 🎉  

---

## 📌 Limitations

- Uses embedded HTML/JS (not pure Streamlit)
- Limited to basic arithmetic operations
- Uses `eval()` in JavaScript (not secure for advanced use)
